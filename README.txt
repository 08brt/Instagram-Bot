# InstaBot
Instagram bot to increase engagement on your profile

REQUIERMENTS
    PYTHON
    SELENIUM
    MACOS (Currently only works on mac as website layout is different on Windows)
    
This bot has a couple main purposes
    1: Automatically logs you into Instagram
    2: Follows and likes pictures in a given hashtag
    3: Follows users from another account
    4: Follows users from another account recent picture
